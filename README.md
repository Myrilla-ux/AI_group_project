#如何使用手机浏览：
（我使用的方法是：）1.手机与电脑处于同一个热点里
                2.电脑终端输入ipconfig getifaddr en0，得到一个地址（我的为172.20.10.2）
                3. 在手机浏览器里输入：http://172.20.10.2:5050   (这里的端口为5050是因为我在app.py中设置的为5050，如果显示端口被占用，可以换一个端口）
